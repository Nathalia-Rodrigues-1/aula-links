Introdução ao MVC

O padrão MVC(Model-View-Controller) é uma arquitetura de software utilizada no desenvolvimento de aplicações para a lógica de negócios , a intergace do usuário e o controle dos dados.


Componentes dp MVC

  • Model(Modelo)- Representa os dados e a lógica de negócios da aplicação.
  • View(Visão)- Responsável pela interface com o usuário e exibição de dados.
  • Controller(Controlador)- Intermedía a comunicação entre Model e View.
