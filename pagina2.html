Model (Modelo) no MVC

O Model representa a camada de dados da aplicação. Ele gerencia a lógica de negócios ,as regras e a interação com o banco de dados.


Prncipais funções do Model

  • Armazena e gerencia os dados.
  • Executa regras de negócio.
  • Interage com o banco de dados.

Exemplo

Imagine um sistema de usuários onde o Model contém as informações como nome ,e-mail e senha além das regras de validação.
